Headings

HTML provides six levels of headings, from h1 (the most important) to h6 (the least important). Headings are used to structure your document and make it more readable.

Example:

This is an h1 heading

This is an h2 heading

This is an h3 heading

This is an h4 heading

This is an h5 heading
This is an h6 heading

Code:

<h1>This is an h1 heading</h1>
<h2>This is an h2 heading</h2>
<h3>This is an h3 heading</h3>
<h4>This is an h4 heading</h4>
<h5>This is an h5 heading</h5>
<h6>This is an h6 heading</h6>

Best Practices:

  • Use only one h1 per page
  • Follow a hierarchical structure
  • Don't skip heading levels
  • Use headings for structure, not styling

Paragraphs

The paragraph element (p) is used to group related content into paragraphs. Browsers automatically add some space before and after each paragraph.

Example:

This is the first paragraph. HTML paragraphs are defined with the p tag. A paragraph always starts on a new line, and browsers automatically add some white space before and after a paragraph.

This is the second paragraph. The browser will automatically remove any extra spaces and lines when the page is displayed.

Code:

<p>
  This is the first paragraph. HTML paragraphs are defined with the p tag.
  A paragraph always starts on a new line, and browsers automatically add some white space before and after a paragraph.
</p>
<p>
  This is the second paragraph. The browser will automatically remove any extra spaces and lines when the page is displayed.
</p>

Note:

Browsers automatically remove extra whitespace and line breaks in HTML. To add a line break within a paragraph, use the br element.

Line Breaks (br)

The br element creates a line break in text. It's useful when you want to break a line without starting a new paragraph. The br element is an empty element, which means it has no closing tag.

Example:

This is a paragraph
with a line break
and another line break.

Code:

<p>
  This is a paragraph<br>with a line break<br>and another line break.
</p>

When to use:

  • Poetry or song lyrics
  • Addresses
  • When you need a line break without starting a new paragraph

Horizontal Rules (hr)

The hr element creates a horizontal rule (line) that represents a thematic break in content. It's often used to separate content sections or indicate a change in topic.

Example:

This is content before the horizontal rule.

This is content after the horizontal rule.

Code:

<p>This is content before the horizontal rule.</p>
<hr>
<p>This is content after the horizontal rule.</p>

Styling with CSS:

You can style the hr element with CSS to change its appearance: 

hr {
  height: 2px;
  background-color: #3b82f6;
  border: none;
}

Superscript and Subscript

The sup element defines superscript text, and the sub element defines subscript text. These are commonly used in mathematical formulas, chemical formulas, and footnotes.

Example:

E = mc2 is Einstein's famous equation.

Water is H2O.

This is the 1st example of a footnote[1].

Code:

<p>
  E = mc<sup>2</sup> is Einstein's famous equation.
</p>
<p>
  Water is H<sub>2</sub>O.
</p>
<p>
  This is the 1<sup>st</sup> example of a footnote<sup>[1]</sup>.
</p>

Common uses:

  • Superscript (sup): Exponents, footnotes, ordinal numbers
  • Subscript (sub): Chemical formulas, mathematical variables

Lists

HTML provides three types of lists: ordered lists (ol), unordered lists (ul), and description lists (dl). Each list item in an ordered or unordered list is defined with the li element.

Unordered List Example:

Shopping List:

  • Milk
  • Eggs
  • Bread
  • Butter

Code:

<h4>Shopping List:</h4>
<ul>
  <li>Milk</li>
  <li>Eggs</li>
  <li>Bread</li>
  <li>Butter</li>
</ul>

Ordered List Example:

Recipe Instructions:

  1. Preheat the oven to 350°F
  2. Mix all ingredients in a bowl
  3. Pour into a baking dish
  4. Bake for 30 minutes

Code:

<h4>Recipe Instructions:</h4>
<ol>
  <li>Preheat the oven to 350°F</li>
  <li>Mix all ingredients in a bowl</li>
  <li>Pour into a baking dish</li>
  <li>Bake for 30 minutes</li>
</ol>

Description List Example:

HTML Terms:

HTML
HyperText Markup Language, the standard language for creating web pages.
CSS
Cascading Style Sheets, used for styling HTML documents.
JavaScript
A programming language that enables interactive web pages.

Code:

<h4>HTML Terms:</h4>
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language, the standard language for creating web pages.</dd>
  
  <dt>CSS</dt>
  <dd>Cascading Style Sheets, used for styling HTML documents.</dd>
  
  <dt>JavaScript</dt>
  <dd>A programming language that enables interactive web pages.</dd>
</dl>

List types:

  • ul: Unordered list (bullet points)
  • ol: Ordered list (numbers or letters)
  • dl: Description list with terms (dt) and descriptions (dd)
