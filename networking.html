Network Types: LAN, WAN, MAN

Computer networks are classified based on their geographical coverage and scale. The three primary types are Local Area Network (LAN), Wide Area Network (WAN), and Metropolitan Area Network (MAN).

Local Area Network (LAN)

A LAN connects computers and devices within a limited area such as a home, school, office building, or campus.

  • Covers a small geographical area
  • High data transfer rates (typically 100 Mbps to 10 Gbps)
  • Low latency and high reliability
  • Usually privately owned and managed

Examples: Office networks, home networks, school computer labs

Wide Area Network (WAN)

A WAN spans a large geographical area, often connecting multiple LANs across cities, countries, or continents.

  • Covers large geographical distances
  • Lower data transfer rates compared to LANs
  • Often uses public telecommunications infrastructure
  • May be managed by multiple organizations

Examples: The Internet, corporate networks spanning multiple locations, banking networks

Metropolitan Area Network (MAN)

A MAN covers a larger area than a LAN but smaller than a WAN, typically spanning a city or large campus.

  • Covers a city or large campus
  • Medium data transfer rates
  • Often owned by a single organization or consortium
  • Connects multiple LANs within a metropolitan region

Examples: City-wide networks, university campus networks, cable TV networks in a city

Network Comparison

Feature LAN MAN WAN
Geographical Area Small (building/campus) Medium (city) Large (country/global)
Speed High (100 Mbps - 10 Gbps) Medium (10 Mbps - 1 Gbps) Varies (56 Kbps - 100+ Mbps)
Ownership Single organization Single or multiple organizations Multiple organizations
Latency Low Medium High

Domain Names

A domain name is a human-readable address that points to a specific location on the internet. It serves as an easier way to access websites instead of remembering IP addresses.

Domain Name Structure

A domain name consists of multiple parts separated by dots:

  • Subdomain (optional): Appears before the main domain (e.g., "blog" in blog.example.com)
  • Second-Level Domain: The main identifier (e.g., "example" in example.com)
  • Top-Level Domain (TLD): The extension (e.g., ".com", ".org", ".net")

Domain Name System (DNS)

The Domain Name System (DNS) is the internet's phonebook. It translates domain names to IP addresses so browsers can load internet resources.

When you enter a domain name in your browser:

  1. Your browser asks a DNS resolver for the IP address
  2. The resolver queries DNS servers to find the IP
  3. The IP address is returned to your browser
  4. Your browser connects to the web server at that IP

This process happens in milliseconds!

Types of Domains

Domain names are categorized into different types based on their top-level domain (TLD). Each type serves different purposes and has different registration requirements.

Generic Top-Level Domains (gTLDs)

These are the most common domain extensions that aren't tied to a specific country.

  • .com - Commercial entities (most popular)

    Example: amazon.com, microsoft.com

  • .org - Non-profit organizations

    Example: wikipedia.org, redcross.org

  • .net - Network organizations (originally)

    Example: cloudflare.net, speedtest.net

  • .edu - Educational institutions

    Example: harvard.edu, mit.edu

  • .gov - U.S. government entities

    Example: whitehouse.gov, nasa.gov

Country Code Top-Level Domains (ccTLDs)

These domains are specific to countries or territories and consist of two letters.

  • .us - United States

    Example: whitepages.us, state.gov.us

  • .uk - United Kingdom

    Example: bbc.co.uk, gov.uk

  • .jp - Japan

    Example: sony.co.jp, go.jp

  • .de - Germany

    Example: bmw.de, bundesregierung.de

  • .id - Indonesia

    Example: kominfo.go.id, ui.ac.id

New gTLDs

Since 2012, hundreds of new generic TLDs have been introduced to provide more options.

  • .app - Mobile applications

    Example: cash.app, notion.app

  • .blog - Blogs and content sites

    Example: wordpress.blog, ghost.blog

  • .shop - E-commerce websites

    Example: apple.shop, nike.shop

  • .tech - Technology companies

    Example: intel.tech, vr.tech

Special Purpose Domains

Some domains serve specific technical or administrative purposes.

  • .int - International organizations

    Example: nato.int, who.int

  • .mil - U.S. military

    Example: army.mil, navy.mil

  • .arpa - Infrastructure purposes

    Example: in-addr.arpa, ip6.arpa

Did you know?

As of 2023, there are over 1,500 TLDs available, with hundreds of millions of registered domain names worldwide!

Web Hosting

Web hosting is a service that allows individuals and organizations to make their websites accessible on the internet. It provides the technologies and services needed for a website to be viewed on the internet.

Shared Hosting

Multiple websites share resources on a single server.

  • Most affordable option
  • Limited resources and performance
  • Good for small websites and blogs
  • Easy to set up and manage

Examples: Bluehost, HostGator, GoDaddy

Price: $3-10/month

VPS Hosting

Virtual Private Server - a virtualized server that acts like a dedicated server within a shared hosting environment.

  • Dedicated resources
  • Better performance than shared hosting
  • Root access to server
  • Good for medium-sized websites

Examples: DigitalOcean, Linode, Vultr

Price: $20-100/month

Dedicated Hosting

An entire physical server dedicated to a single website or application.

  • Maximum performance and resources
  • Complete control over server configuration
  • Highest level of security
  • Best for high-traffic websites

Examples: OVHcloud, Hetzner, GoDaddy

Price: $80-500+/month

Cloud Hosting

Hosting that uses multiple servers in a cloud network to balance load and maximize uptime.

  • Highly scalable resources
  • Pay-as-you-go pricing model
  • High reliability and uptime
  • Good for websites with variable traffic

Examples: AWS, Google Cloud, Microsoft Azure

Price: Varies based on usage

Managed WordPress Hosting

Specialized hosting optimized specifically for WordPress websites.

  • WordPress pre-installed and optimized
  • Automatic updates and backups
  • Enhanced security features
  • Expert WordPress support

Examples: WP Engine, Kinsta, Flywheel

Price: $25-300+/month

Choosing the Right Hosting

When selecting a hosting provider, consider these factors:

  • Performance needs - Consider expected traffic and resource requirements
  • Budget - Balance cost with necessary features
  • Technical expertise - Choose managed solutions if you lack technical skills
  • Scalability - Ensure the hosting can grow with your website
  • Uptime guarantees - Look for 99.9% or higher
  • Support quality - 24/7 support is essential for business websites
  • Security features - SSL certificates, firewalls, backups
  • Control panel - User-friendly interface like cPanel or Plesk
